这就用到escape了。

比如： 找出名字中含有「%」的论文：

escape的意思是：后面的内容\是前面字符串%\%%的转义字符。这样前面字符串中的第2个%将不再认为是通配符，而被认为是要匹配的内容。

escape 只作用于一个like，当出现多个like都需要转义时，每一个like后面都需要加escape。
